Image

Images are the most important component of a website as they attract the user's attention and, therefore, should be responsive. We offer the following types of images - Responsive images, Images rounded corner, and Round Images.

Responsive Image

You can use this responsive image in background or any of the web page. It will enhance its beauty. To use it include img-responsive class as shown below.

Images with Round Corners

To generate an image with rounded cornders, add the base class img-responsive along with the supporting class round-corner.

Round Images

To generate a round image, use the base class img-round along with the base class img-responsive.
