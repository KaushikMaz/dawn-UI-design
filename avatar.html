Avatar

Avatars displays user image or initials as profile picture. Avatars can be used for 'container' objects — projects, spaces, repositories, etc. — to give them a recognisable visual identity.

Here we offer the following range of avatars - Circle Avatars, Square Avatars, and Text Avatars in 5 different sizes - extra small, small, medium, large, and extra large, along with the Avatar of status.

Circle shaped Avatar

To use Circle shaped Avatars,You need to include class avatar and for size add class according to size avatar-lg, avatar-md, avatar-sm, avatar-xs (e.g. class="avatar avatar-lg") as shown below.
